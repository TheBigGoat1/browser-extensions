How to Get API Keys

  1. Log in to Binance (or Testnet for Paper).
  2. Go to API Management: Binance API Management (Testnet: Profile → API Management).
  3. System generated — Choose “Create API” → “System generated”.
  4. Secret only shows once — Copy and save the Secret immediately; Binance will not show it again.
  5. API restrictions (recommended): Enable Reading, Enable Spot & Margin Trading, Enable Futures. Do not enable Withdrawals.
  6. Paste API Key and Secret above. Values are kept when you click out to copy (session draft).

Parameter definitions & limits

  • Symbol — Trading pair (e.g. BTCUSDT). Must exist on Binance; invalid pair shows red.
  • Notional — Order size in quote currency. Minimum per pair (e.g. $5).
  • Leverage — 1–125; max depends on symbol. Wrong leverage for pair shows red. Liquidation % = 100 ÷ leverage.
  • Callback rate — Binance accepts only 0.1% increments (0.1, 0.2, 0.3…). 0.15% will error; use 0.1 or 0.2.
  • Stop loss / Trailing — Initiation % from entry; levels move SL at profit triggers. Trailing: profit_pct and callback_rate_pct in 0.1% steps.
