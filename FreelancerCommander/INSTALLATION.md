## **Step-by-Step Setup**

### **1. Load the Extension**
1. Open Chrome browser
2. Navigate to `chrome://extensions`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Select the `FreelancerCommander` folder
6. Extension should appear in your extensions list

### **2. First-Time Configuration**
1. Click the extension icon in your toolbar
2. The side panel will open with 3 AI panels
3. **Log into each AI service:**
   - ChatGPT: Log in at chatgpt.com
   - Claude: Log in at claude.ai
   - DeepSeek: Log in at chat.deepseek.com
4. Once logged in, you're ready to use!

### **3. Start Monitoring**
1. Open Freelancer.com in a tab
2. Navigate to job listings
3. The extension automatically starts monitoring (every 30 seconds)
4. You'll get notifications when new jobs are detected

### **4. Using the Command Center**

#### **Method 1: Context Menu (Recommended)**
1. Highlight job description text on Freelancer.com
2. Right-click â†’ Select **"Send to AI Command Center"**
3. Side panel opens automatically
4. Click action button (Translate/Proposal/Urgent)
5. Prompt is copied to clipboard
6. Paste (Ctrl+V) into your chosen AI panel

#### **Method 2: Manual Selection**
1. Highlight text on any page
2. Open side panel manually
3. Click action button
4. Paste into AI panel

## **Troubleshooting**

### **Issue: AI Panels Show Blank**
**Solution:**
- Make sure you're logged into each AI service
- Try clicking "Refresh All" button
- Check browser console (F12) for errors
- Some AI sites may block iframes - this is normal, try refreshing

### **Issue: Jobs Not Being Detected**
**Solution:**
- Ensure Freelancer.com tab is open and active
- Wait 30 seconds for first check
- Check that you're on a job listing page (not profile/settings)
- Open DevTools â†’ Console â†’ Check for errors

### **Issue: Context Menu Not Appearing**
**Solution:**
- Reload the extension
- Check permissions in `chrome://extensions`
- Try selecting text and right-clicking again
- Restart Chrome if needed

### **Issue: Extension Not Loading**
**Solution:**
- Check `manifest.json` for syntax errors
- Ensure all files are in the same folder
- Check Chrome console for specific errors
- Try removing and re-adding the extension
