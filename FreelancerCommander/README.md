## üì¶ **Installation**

### **Step A:** Load Extension
1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the `FreelancerCommander` folder

### **Step B:** First-Time Setup
1. Click the extension icon to open the side panel
2. Log into your AI accounts (ChatGPT, Claude, DeepSeek) in the iframes
3. The extension will automatically start monitoring Freelancer.com

### **Step C:** Start Using
1. Open Freelancer.com in a tab
2. Highlight a job description
3. Right-click ‚Üí **"Send to AI Command Center"**
4. Click action buttons in the side panel

## üêõ **Troubleshooting**

### **AI Panels Show Blank**
- Check if you're logged into the AI services
- Try refreshing the panels (click "Refresh All")
- Check browser console for errors
- Ensure rules.json is loaded correctly

### **Jobs Not Detected**
- Make sure Freelancer.com tab is open
- Check that you're on a job listing page
- Wait 30 seconds for first check
- Check background script console for errors

### **Context Menu Not Appearing**
- Reload the extension
- Check if context menu permission is granted
- Try right-clicking on selected text again
